Privacy Policy

Last updated: January 2025

Application: Email Automation Tool

1. Introduction

This Privacy Policy describes how Email Automation Tool ("the Application") collects, uses, and protects information obtained through the Gmail API.

2. Data We Access

The Application accesses the following data from your Google account:

  • Email metadata (sender, subject, date)
  • Email content (body text)
  • Email attachments (files attached to emails)

3. How We Use the Data

Data is used exclusively for:

  • Monitoring incoming emails for automated workflow processing
  • Extracting attachments (Excel and ZIP files) for internal business data processing
  • No data is used for advertising, marketing, or any purpose other than the stated functionality

4. Data Storage and Security

  • All data is processed on private, self-hosted servers
  • No data is stored in third-party cloud services
  • Data is retained only as long as necessary for processing
  • Access to servers is restricted and protected

5. Data Sharing

We do not share, sell, or transfer any user data to third parties. This is a private internal tool.

6. Google API Services User Data Policy

This Application's use and transfer of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.

7. Data Deletion

Users can request complete deletion of any processed data by contacting us. Upon receiving a deletion request, all associated data will be permanently removed within 30 days.

8. Revoking Access

You can revoke this application's access to your Google account at any time by visiting: Google Account Permissions

9. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted on this page with an updated date.
